基本

  1. 安装 ubuntu 22.04 版本系统
  2. 配置 .ssh 文件（复制备份）可用于远程登录
  3. 安装基本程序和编译器，包括
    1. git
    2. gcc
    3. make
    4. zsh
    5. vim
    6. curl
    7. wget
  4. 终端外观调整，并添加 omz
    1. 确保已经安装 git 和 zsh 并且已经拥有 ssh-key.
    2. 获取安装脚本并安装
      sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
    3. 配置 .zshrc 文件
    4. 安装插件
      git clone https://github.com/zsh-users/zsh-autosuggestions.git
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git
    5. 重新启动机器，使 zsh 生效
  5. 安装 OpenJDK
  6. 安装 Edge, vscode. 安装 vscode 插件，包含项目管理，背景自定义，rosely light 主题，图标，gitlens 以及 python 的捆绑开发工具，最后是中文界面。
  7. 安装 intelliJ IDEA.

01 窗口行为调整

  1. 安装 gnome-tweaks | sai gnome-tweaks
  2. 通过 gnome-tweaks 使新建窗口居中
  3. 前往 https://extensions.gnome.org/ 安装浏览器插件
  4. 选择需要的插件

02 输入法配置 | 参考中州韵输入法设置

  1. install ibus-rime: sudo apt-get install ibus-rime
  2. open ibus-rime: 重新啓動，在鍵盤-輸入法中添加 Chinese-ibus-rime. 如果找不到該選項，多輸入幾次。成功啓用後，進入 ~/.config/ibus 目錄，可以看到 rime 文件夾, 這個文件夾將用於以後配置。
  3. 使用 东风破 管理工具快速配置雾凇词库

bash rime-install iDvel/rime-ice:others/recipes/full

02 docker 安装

  1. Update the package index and install packages to allow to use a repository over HTTPS:
sudo apt-get update
sudo apt-get install ca-certificates curl gnupg
  1. Add Docker’s official GPG key:
sudo install -m 0755 -d /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
sudo chmod a+r /etc/apt/keyrings/docker.gpg
  1. Use the following command to set up the repository:
echo \
  "deb [arch="$(dpkg --print-architecture)" signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu \
  "$(. /etc/os-release && echo "$VERSION_CODENAME")" stable" | \
  sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
  1. Install docker engine:
sudo apt-get update
sudo apt-get install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
sudo docker run hello-world
