基本

  1. 安装 ubuntu 22.04 版本系统
  2. 配置 .ssh 文件（复制备份）可用于远程登录
  3. 安装基本程序和编译器，包括
    1. git
    2. gcc
    3. make
    4. zsh
    5. vim
    6. curl
    7. wget
  4. 终端外观调整，并添加 omz，细节请参考附录中的 ubuntu 命令行美化
  5. 安装 OpenJDK
  6. 安装 Edge, vscode. 安装 vscode 插件，包含项目管理，背景自定义，rosely light 主题，图标，gitlens 以及 python 的捆绑开发工具，最后是中文界面。
  7. 安装 intelliJ IDEA.

进阶

完成 alias 配置

01 窗口行为调整

  1. 安装 gnome-tweaks | sai gnome-tweaks
  2. 通过 gnome-tweaks 使新建窗口居中
  3. 前往 https://extensions.gnome.org/ 安装浏览器插件
  4. 选择需要的插件

02 输入法配置 | 参考中州韵输入法设置

  1. install ibus-rime: sudo apt-get install ibus-rime
  2. open ibus-rime: 重新啓動，在鍵盤-輸入法中添加 Chinese-ibus-rime. 如果找不到該選項，多輸入幾次。成功啓用後，進入 ~/.config/ibus 目錄，可以看到 rime 文件夾, 這個文件夾將用於以後配置。
  3. 使用 东风破 管理工具快速配置雾凇词库
  1. 全部完成！
